# zichainTestTask

## Running the app

```bash
# install packages
$ npm i

# run eslinter
$ npm run eslint

# run apitests
$ npm run apitests

# run application
$ npm start

follow http://0.0.0.0:3000/link/add post request with req.body.usersLink 
to add long link and get short one

follow http://0.0.0.0:3000/* to be moved to long link
```
